# vastum
Your Life in Numbers


erstelle eine website. html und js.
Name: Vastum - Dein Leben in Wochen.

Die website visualisiert ein menschenleben in punkten, jeder punkt eine Woche. verschiedene auswertungen werden unterschiedlich visualisiert. 

standard ansicht.
zum start werden eingafelder anzeigt die im local storage gespeichert werden. die Angaben können jederzeit verändert werden und werden über ein Menu/Einstellungen symbol rechts oben geöffnet und geschlossen.
Die Angaben sind:
"Vorname", Geschlecht, Alter heute (errechnet Geburtsjahr), wohnort, Beruf, schlaf, handynutzung hh:mm pro Tag, sport stunden pro Woche, freunde stunden pro Woche, tv hh:mm pro Tag, rauchen stück pro woche, süßigkeiten, körpergröße, gewicht, alkohol nie/selten/regelmäßig - liter pro woche, blutdruck niedrig/normal/hoch, stress niedrig/normal/hoch, arbeitsstunden pro woche, urlaubstage pro jahr, krankheitstage pro jahr, gehalt, vermögen,

mache ein button für beispielwerte. dann werden alle werte mit einer beispielperson ausgefüllt.

nach dem "Fertig" drücken der Angaben kommt man zur Auswertung. 

in der Mitte sind in einem rechteck punkte für jede Woche des eingegebenen lebens. ausgefüllte punkte für bereits vergangene wochen und leere helle punkte die zu erwarten sind.

rechts daneben stehen die sinnvollen interessanten fakten in zahlen. und statistische auswertungen. vergangene wochen, künftige wochen, anteilig verbrauchte wochen.

darüber sind auswahlpunkte: Schlaf, social media, freunde, arbeit, sport,...
beim klick wird die wochengrafik so ausgefüllt, dass die zeiten in jeweils unterschiedlichen farben über die vergangenen wochen gelegt werden, zur visualisierung wie viel vom leben man mit der ausgewählten tätigkeit verbraucht hat. 

zusätzlich kann man die Ansicht ändern von "wochenansicht gesamt" zu einzelansicht. Dann wird von links nach rechts werden alle sinnvollen auswahlpunkt angezeigt und darunter in rechtecken wie viele wochen jemand im gesamtleben mit dieser beschäftigung zugebracht hat.
